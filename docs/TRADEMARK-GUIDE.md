## Step 1: Preliminary Trademark Search

Before filing, conduct a thorough search to ensure your marks are available.

### Germany (DPMA)

1. **Visit**: [DPMA Register](https://register.dpma.de/DPMAregister/marke/einsteiger)
2. **Search Type**: "Markenrecherche" (Trademark Search)
3. **Search Terms**: Enter each mark (e.g., "MOBAflow", "MOBA")
4. **Nice Classes**: Focus on:
   - **Class 9**: Computer software, mobile apps
   - **Class 42**: Software development services
   - **Class 28**: Games, toys (model railways)

### European Union (EUIPO)

1. **Visit**: [EUIPO eSearch plus](https://euipo.europa.eu/eSearch/)
2. **Search Type**: Basic search
3. **Territory**: European Union
4. **Search**: Enter each trademark
5. **Check**: Similar marks in phonetic/visual similarity

### International (WIPO Global Brand Database)

1. **Visit**: [WIPO Global Brand Database](https://branddb.wipo.int/)
2. **Search**: Enter trademark names
3. **Filter**: By territory (DE, EU, US, etc.)
4. **Review**: Similar existing registrations

## Step 2: Nice Classification

Trademarks are registered in specific "Nice Classes". For software:

| Class | Description | Relevant Products |
|-------|-------------|-------------------|
| **9** | Computer software, downloadable apps, electronic publications | MOBAflow, MOBAsmart, MOBAdash |
| **28** | Games and playthings; model vehicles | Model railway control systems |
| **42** | Scientific and technological services; software design | SaaS, cloud services |

**Recommendation**: Register in Classes 9 and 42 minimum. Add Class 28 if targeting model railway retail.

## Step 3: Registration Process

### Germany (DPMA)

| Item | Details |
|------|---------|
| **Authority** | Deutsches Patent- und Markenamt (DPMA) |
| **Website** | [dpma.de](https://www.dpma.de/marken/) |
| **Online Filing** | [DPMAdirektWeb](https://direktweb.dpma.de/) |
| **Fees** | EUR 290 (base, up to 3 classes) + EUR 100 per additional class |
| **Processing Time** | 3-6 months |
| **Validity** | 10 years (renewable) |

**Steps**:
1. Create DPMA account at DPMAdirektWeb
2. File "Markenanmeldung" (trademark application)
3. Upload trademark (word mark = just text)
4. Select Nice classes
5. Pay fees via bank transfer
6. Wait for examination
7. Receive registration certificate

### European Union (EUIPO)

| Item | Details |
|------|---------|
| **Authority** | European Union Intellectual Property Office |
| **Website** | [euipo.europa.eu](https://euipo.europa.eu/) |
| **Online Filing** | EUIPO User Area |
| **Fees** | EUR 850 (1 class) + EUR 50 (2nd class) + EUR 150 (each additional) |
| **Processing Time** | 4-6 months (if no opposition) |
| **Validity** | 10 years (renewable) |
| **Coverage** | All 27 EU member states |

**Steps**:
1. Create EUIPO User Area account
2. File EU Trade Mark (EUTM) application
3. Complete Fast Track (pre-approved terms) for speed
4. Pay fees online
5. Examination period (1 month)
6. Publication period (3 months for opposition)
7. Registration

### International (WIPO Madrid System)

For protection beyond EU (e.g., USA, China, Japan):

| Item | Details |
|------|---------|
| **Authority** | World Intellectual Property Organization |
| **System** | Madrid Protocol |
| **Prerequisite** | Must have national/EU registration first |
| **Fees** | CHF 653 (base) + country-specific fees |
| **Coverage** | 130+ countries |

## Step 5: Post-Registration

### Trademark Notice

Add to all products, documentation, and marketing:

```
MOBAflow™, MOBAsmart™, MOBAdash™, MOBAerp™, MOBAixl™, and MOBAtps™ 
are trademarks of [Your Name/Company].
```

After registration, use ® instead of ™.

### Monitoring

- Set up alerts on DPMA/EUIPO for similar filings
- Consider trademark watch service
- Document first use dates

### Renewal

- DPMA: Every 10 years, EUR 750 (3 classes)
- EUIPO: Every 10 years, EUR 850 (1 class)
